# weibo_image_uploader

一个利用新浪微博作图床单的轮子。

```
#运行
./weibo_image_uploader -nickname=水印文案 -account=新浪微博账号 -password=新浪微博密码 -url=图片地址（http开头，或者本地文件）
```
